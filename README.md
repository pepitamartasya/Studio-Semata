# Studio Semata Website

Landing page statis untuk Studio Semata. Tidak memerlukan database atau biaya hosting.

## Preview lokal

Buka `index.html` langsung di browser, atau jalankan server statis dari folder ini.

## Publish gratis

Upload isi folder ini ke GitHub Pages, Cloudflare Pages, atau Netlify. Semua tombol konsultasi sementara diarahkan ke Linktree Studio Semata.
